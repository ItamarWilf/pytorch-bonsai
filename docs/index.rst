Pytorch-Bonsai Documentation
============================

:mod:`bonsai` is a high-level library to help with pruning neural networks in PyTorch.



Installation
============

From pip:

.. code:: bash

    pip install pytorch-bonsai


From source:

You can clone the source from `github <https://github.com/ItamarWilf/pytorch-bonsai>`_ and run

.. code:: bash

    python setup.py install
